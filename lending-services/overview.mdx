<Info>
Perch Lending Services is currently in development and not yet available for external integration. [Contact us](mailto:partnersupport@myperch.io) to learn more or request early access.
</Info>

Perch Lending Services is a separate API platform from Perch Core, purpose-built for real-time mortgage decisioning. It will expose auto-adjudication endpoints, live rate retrieval, and mortgage product search — and will power the next generation of Perch embeds including the upcoming Mortgage Search widget.

Full documentation will be published here when the API is available.
