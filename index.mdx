## Getting Started

<Steps>
  <Step title="Read the API Overview">
    Start with our [API Overview](/getting-started/api-overview) to understand authentication, terms of use, and key concepts.
  </Step>
  <Step title="Choose Your API">
    Select the appropriate API for your use case:
    - **Partner API**: For lead management and partner workflows
    - **Realtor API**: For realtor portal functionality
  </Step>
  <Step title="Get Your API Key">
    Contact your Perch representative to obtain API credentials for your integration.
  </Step>
  <Step title="Start Building">
    Use our interactive API playground to test endpoints and build your integration.
  </Step>
</Steps>

## API Environments

<Note>
  All APIs are available in both development and production environments with different base URLs.
</Note>

- **Development**: For testing and development
- **Production**: For live integrations
