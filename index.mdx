# Perch Developer Platform

Welcome to the **Perch Developer Platform** — a suite of APIs and tools that enable partners to deliver exceptional digital mortgage experiences for their clients.

## Core API

The Core API gives partners and realtors programmatic access to Perch's mortgage platform — submit leads, track applications, and stay in sync as clients move through the mortgage lifecycle.

## Lending API

The Lending API is a separate decisioning platform that exposes real-time rate data, auto-adjudication, and mortgage product search. It will power the next generation of Perch embeds including the upcoming Mortgage Search widget.
